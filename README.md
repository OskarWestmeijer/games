# Vuodenkierto: Vanha maatila

A cosy **walking sim** set on a 17th-century Finnish farm — a quiet homestead by
the water. Wander, watch the light on the lake, listen. No combat, no inventory, no
quests. Inspired by *Songs of Glimmerwick* and *Pentiment*.

This is an early proof-of-concept. See [CLAUDE.md](CLAUDE.md) for the full brief,
architecture, and roadmap.

## Run

```bash
npm install
npm run dev      # then open the printed http://localhost URL in Chrome
```

Walk with **WASD** or the **arrow keys**.

## Stack

Vite + TypeScript + HTML5 Canvas 2D. No game framework.
